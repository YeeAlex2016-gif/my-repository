# 王宇峰｜個人網頁作品集

這是一個可以直接上傳到 GitHub Pages 的靜態個人網站專案，使用：

- HTML
- CSS
- JavaScript

## 檔案內容

```text
index.html   # 網站主頁
style.css    # 網站樣式
script.js    # 選單、作品篩選、捲動動畫
README.md    # 說明文件
```

## 如何修改個人資料

請打開 `index.html`，搜尋以下內容並替換：

```html
your.email@example.com
https://github.com/yourname
https://www.linkedin.com/in/yourname
```

也可以替換作品名稱、作品描述、技能、履歷檔案連結等內容。

## 發佈到 GitHub Pages

1. 到 GitHub 建立一個新的 Repository，例如：`personal-website`
2. 上傳本專案所有檔案到 Repository 根目錄
3. 進入 Repository 的 `Settings`
4. 點選左側 `Pages`
5. Source 選擇 `Deploy from a branch`
6. Branch 選擇 `main`，資料夾選擇 `/root`
7. 按下 `Save`
8. 等待 1～3 分鐘後，GitHub 會提供網站網址

## 注意

如果你想使用「下載履歷」按鈕，請把你的履歷 PDF 命名為 `resume.pdf`，並放在和 `index.html` 同一層。
